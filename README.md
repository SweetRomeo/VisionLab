# VisionLab

VisionLab is a research-oriented image enhancement benchmark platform developed with C++, Qt, OpenCV, Java, and Spring Boot.

The main goal of this project is to evaluate image enhancement algorithms for autonomous vehicle vision systems by comparing both visual quality and computational performance.

## Project Goals

- Implement image enhancement algorithms using C++ and OpenCV
- Build a Qt-based desktop application for visual comparison
- Develop a Spring Boot REST API to store benchmark results
- Compare algorithm performance using metrics such as execution time, PSNR, and SSIM
- Support master's thesis research on image enhancement for autonomous vehicle vision systems

## Technologies

- C++
- OpenCV
- Qt
- Java
- Spring Boot
- PostgreSQL
